O elemento HTML é usado para representar a raiz de um HTML ou XHTML documento. Todos os outros elementos devem ser descendentes desse elemento.

Por exemplo a tag LINK especifica relações entre o documeento atual e um recurso externo, é mais usado para vincular as folhas de estilo.

Já a tag META define qualquer informação de metadados que não podem ser definidos por outros elementos como (base, link, style, title).

A tag STYLE contém informações de estilo para um documento ou para parte deste documento. Informações de estilo estão contidas nesse elemento.
