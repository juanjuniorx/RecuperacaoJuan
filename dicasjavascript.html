Dicas de uso JavaScript

 O bloco else é usado caso apenas uma condição precise ser avaliada.

O comando switch veifica o valor de uma variável, para cada um das opções e executa um conjunto de ações. Caso nenhum dos valores seja verificado, os comandos do bloco default são executados

A estrutura de repetição while é utilizada para executar um conjunto de ações enquanto uma condição for verdadeira. Se a condição se tornar falsa esse bloco é finalizado.

Sintaxe da estrutura for por exemplo se uma variável contador é inicilizada como o valor zero, e enquanto for menor que 10, então deve ser executado

E se for centralizar um texto use a tag text-align: center que ficará centralizado, corretamente como deseja.
