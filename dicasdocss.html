Dicas de uso CSS

 Use o "Inspetor" ele é usado para adicionar ou editar classes de CSS em tempo real, testar animações em tempo real, adicionar qualquer tipo de comando CSS que o navegador suporte.

Se você quer resetar o CSS use a tag "Css Reset" , isso fará a função de "limpar casa".

Caso queira mudar a cor do texto inserido em CSS use a tag color e na frente atribua uma cor a ela, exemplo irei escolher a cor laranja então o código ficará assim: color: orange; teste123

Para mudar a cor de fundo da página é só usar background-color na frente de color você irá escolher uma cor para sua página.

E se for centralizar um texto use a tag text-align: center que ficará centralizado, corretamente como deseja.
